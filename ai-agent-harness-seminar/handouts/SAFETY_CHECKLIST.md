## 실행 통제

- [ ] Sandbox가 파일·프로세스·네트워크 중 무엇을 실제로 강제하는지 확인했다.
- [ ] 승인 지점과 승인자가 명시되어 있다.
- [ ] Agent가 Approval Gate를 우회할 Tool이 없다.
- [ ] 입력 원본은 읽기 전용 또는 복사본이다.
- [ ] 출력 경로가 제한되어 있다.
- [ ] 삭제와 비가역 행동은 기본 금지다.
- [ ] 비용·시간·재시도 한도가 있다.
- [ ] 비상 중단 방법을 리허설했다.

## 검증과 복구

- [ ] Validator가 Agent와 독립적으로 실행된다.
- [ ] Validator가 검사하지 않는 영역을 기록했다.
- [ ] Stop Condition이 구체적이다.
- [ ] Trace에 Tool 호출, 결과, 오류, 승인, 중단이 남는다.
- [ ] Trace 보존 기간과 접근 권한이 정해졌다.
- [ ] Rollback 또는 깨끗한 복사본 복원이 가능하다.
- [ ] 실패 시 사람이 확인할 Artifact가 남는다.
- [ ] 실제 장비·생산 시스템과 분리된 환경에서 먼저 검증한다.
