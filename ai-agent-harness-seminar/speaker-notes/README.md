# Speaker Notes 운영

샘플 슬라이드의 마지막 HTML comment는 Slidev Presenter Note로 사용한다. 본편 작성 시 화면 문구를 그대로 읽지 말고 다음을 노트에 둔다.

- 이 슬라이드의 확인된 사실과 출처
- 잠정 가정과 철회 조건
- 비유의 한계
- 질문을 받을 가능성이 높은 용어
- 다음 슬라이드로 넘길 기준
- 데모 실패 시 대체 Trace 전환 문장

Presenter Mode 기능 근거: Slidev Syntax Guide, https://sli.dev/guide/syntax, 확인 2026-07-21. 제품 기능은 발표 직전 다시 확인한다.
