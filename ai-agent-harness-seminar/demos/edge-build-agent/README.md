# 합성 Build/Test Failure 조사 데모

이 디렉터리는 향후 보조 데모를 구현할 경계다. 현재는 합성 코드 fixture, 실패 테스트, Agent 실행 코드가 없다.

- 실제 회사 코드와 로그를 사용하지 않는다.
- 중첩 Git 저장소를 만들지 않는다.
- 실패 테스트를 재현한 뒤 허용 구현 파일만 최소 수정한다.
- 테스트 삭제·완화, 패키지 설치, 네트워크, 허용 경로 밖 수정은 승인 없이는 금지한다.
- 완료는 테스트와 Diff Validator가 판정한다.

구현 도구와 실행 환경은 미확정이다. 자세한 설계는 docs/DEMO_PLAN.md를 따른다.
